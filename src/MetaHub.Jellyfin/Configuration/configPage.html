MetaHub

Aggregated metadata and artwork for movies, series, anime, music and books — running fully inside Jellyfin (no server, no Docker).

General Libraries Metadata sources Help

Mode

Everything runs inside Jellyfin with a local database. Disable only if you operate a separate MetaHub server.
e.g. http://localhost:8080

Content types

MetaHub only provides metadata and artwork for the selected types.

Language

Two-letter code (de, en, ja, …). Used for descriptions and artwork selection.
Used when the preferred language is not available.
Toggle MetaHub on or off for individual libraries, series, seasons or movies. Children inherit a disabled parent. Changes are saved automatically.

Enrichment

"Keep" never changes data you already have; genres and artwork are always added, never removed.
Enrich an item right when it is first matched, instead of waiting for the scheduled task.

API keys (optional)

Enables movie/series data from TMDB. Free key at themoviedb.org → Settings → API. Without a key, TMDB is simply skipped.
Optional, improves book lookups. Without a key, Open Library is used.
Adds Japanese titles and artwork from annict.com, the Japanese anime database. Free personal token at annict.com → Settings → Applications. Without a token, Annict is skipped. Tip: set the preferred language to ja to also get Japanese episode titles.

AniDB file identification

Identifies anime files exactly by file hash (the Shoko principle) instead of guessing from file names. Requires a free AniDB account and a registered UDP client (anidb.net → Settings → Clients). AniDB enforces strict rate limits — MetaHub queries it slowly and caches every answer permanently.

First steps

  1. Check General — embedded mode on, pick your content types and language.
  2. Open Dashboard → Scheduled Tasks and run MetaHub: Update anime mappings once (downloads the cross-reference datasets, ~35k anime).
  3. Then run MetaHub: Enrich metadata — it fills in descriptions, genres and artwork.
  4. In your library settings, enable MetaHub as a metadata/image provider and refresh metadata on a title to see it work.

Good to know

  • The database lives in Jellyfin's data folder (metahub/metahub.db).
  • Both tasks re-run on a schedule automatically (weekly / daily by default) — adjust them in Scheduled Tasks.
  • Use the Libraries tab to exclude individual libraries, series, seasons or movies.
  • MetaHub also serves seasons and episodes — enable it under the season/episode metadata downloaders and image fetchers in your library settings.
