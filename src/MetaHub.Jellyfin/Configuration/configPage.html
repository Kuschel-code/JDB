MetaHub aggregates cross-linked metadata and artwork for music, movies, series, anime and books.

In embedded mode everything runs inside Jellyfin with a local SQLite database; datasets are fetched from GitHub. Run MetaHub: Update anime mappings then MetaHub: Enrich metadata from Dashboard → Scheduled Tasks.
